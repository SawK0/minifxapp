# minFX ⚡

A modern collection of developer utilities built with Angular. Fast, simple, and privacy-focused tools for everyday development tasks.

## 🌐 Live Demo

**[View Live Demo →](https://minifxapp.vercel.app/)**

> 🚀 Deployed on Render with automatic updates from the main branch

## ✨ Features

- **Privacy First** - All processing happens locally in your browser
- **Modern UI** - Clean, responsive interface built with Angular 21
- **Fast & Lightweight** - Optimized performance with server-side rendering
- **Developer Focused** - Tools designed by developers, for developers

## 🚀 Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm (v11.6.2 or higher)

### Installation

```bash
# Clone the repository
git clone https://github.com/SawK0/minifxapp.git

# Navigate to project directory
cd minifxapp

# Install dependencies
npm install
```

### Development

Start the development server:

```bash
npm start
```

Navigate to `http://localhost:4200/`. The application will automatically reload when you make changes.

### Build

Build the project for production:

```bash
npm run build
```

The build artifacts will be stored in the `dist/` directory.

### Server-Side Rendering

Run the SSR server:

```bash
npm run serve:ssr:minifxapp
```

## 🛠️ Tech Stack

- **Framework**: Angular 21.2
- **Language**: TypeScript 5.9
- **Styling**: SCSS with CSS custom properties
- **Icons**: Material Icons
- **SSR**: Angular Universal with Express
- **Testing**: Vitest
